How to decode a Taylor serial number

A quick manual method when you don't want to use a decoder.

Taylor guitar serial numbers are found on the back of the headstock or on a label inside the soundhole, with formats changing over time. Generally, for modern guitars (post‑2000), the 11‑digit code reveals the year, month, day, and production sequence, while pre‑2000 models use 9 digits.

11‑Digit Serial Numbers (2000–2009)

Example: 20070311301

  • Digits 1–4: Year (e.g., 2007)
  • Digits 5–6: Month (e.g., 03 = March)
  • Digits 7–8: Day (e.g., 11 = 11th)
  • Digit 9: Series code (0 = 300/400, 1 = 500+ Series, 2 = 200, 8 = 100)
  • Digits 10–11: Production sequence

9‑Digit Serial Numbers (1993–1999)

Example: 980103004

  • Digits 1–2: Year (e.g., 98 = 1998)
  • Digits 3–4: Month (e.g., 01 = January)
  • Digits 5–6: Day (e.g., 03 = 3rd)
  • Digit 7: Series code (0 = 300/400, 1 = higher)
  • Digits 8–9: Sequence

Post‑November 2009 & Current System

  • 10 Digits: The format changed to 10 digits. The first digit represents the factory (1 = El Cajon, 2 = Tecate, Mexico).
  • Digits 2–5: Year of production.
  • Digits 6–9: Date of production (MM/DD).
  • Digit 10: Daily production sequence.

Taylor Model Number Key

  • 1st Digit: Series (e.g., 3xx = 300 Series).
  • 2nd Digit: String count/top wood (1 = 6‑string/softwood, 2 = 6‑string/hardwood, 5 = 12‑string/softwood, 6 = 12‑string/hardwood).
  • 3rd Digit: Body Shape (e.g., 0 = Dreadnought, 2 = Grand Concert, 4 = Grand Auditorium, 6 = Grand Symphony).
  • Suffix: “c” = cutaway, “e” = electronics.
  • Wood Variant: “-R” = Rosewood, “-K” = Koa.
